# Weak Areas Log

A running list of every concept you've gotten wrong, found confusing, or had to re-read. Drives the **Adaptive Weak-Area Drill** (`../simulations/adaptive-weak-area-drill.md`).

## How to use this file

1. Whenever you miss a quiz/mock question, add a one-line entry here with:
   - **Concept** (verbatim from the question explanation)
   - **Domain** (Logging Fundamentals · Log Collection · …)
   - **Status** (`open` / `reviewed` / `mastered`)

2. Aim for **mastered** status before the exam. A topic is mastered when you can:
   - Explain it in plain language to an imaginary colleague.
   - Recall the relevant Datadog feature names without looking.
   - Get the next two related questions right.

3. Re-visit weekly: re-explain each `open` item, mark mastered if confident, otherwise schedule a drill.
