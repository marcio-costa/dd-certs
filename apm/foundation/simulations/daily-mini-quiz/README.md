# Daily Mini-Quiz (10 Qs) — Pool

> A **rotating pool** of 10-question warm-ups. Pull a fresh set each day to keep your retrieval muscle warm without sinking 30+ minutes.

## How to use

1. Pick the next file in numerical order (or invoke the cert-prep skill with **"daily mini-quiz"** for a fresh randomized one).
2. Time yourself: aim for 10 minutes total, ~1 min per question.
3. Score, then review missed questions against the linked KB section.
4. Log in `progress/daily-tracker.md`.

## Files

- [`mini-quiz-01.md`](mini-quiz-01.md) — Mixed (all domains)
- [`mini-quiz-02.md`](mini-quiz-02.md) — Mixed (all domains)
- [`mini-quiz-03.md`](mini-quiz-03.md) — Mixed (all domains)

If you've burned through all three, ask the skill to **"generate today's daily mini-quiz"** and it'll create a fresh one from the KB.

## Rules

- Single-answer multiple choice only (matches the real exam format).
- No looking at the KB during the quiz.
- After scoring: re-read the KB section for any missed question.
