# Full Mock Exam — 75 Qs, 2 hours

> The closest possible simulation of the real **APM & Distributed Tracing Fundamentals** exam.

## Format

- **75 scored multiple-choice questions** (single correct answer).
- **120 minutes** total seat time.
- Domain weighting matches the official content outline:
  - Domain 1 (APM Fundamentals): 19 Qs (~25%)
  - Domain 2 (Application Instrumentation): 22 Qs (~30%)
  - Domain 3 (Insight Discovery): 22 Qs (~30%)
  - Domain 4 (Troubleshooting Application using APM): 12 Qs (~15%)
- Difficulty mix: ~30% easy / ~50% medium / ~20% hard.
- All questions are single-answer (matches the real exam — no select-N, no ordering).

## Rules to mirror real conditions

- **Answer in one sitting.** No bathroom breaks, no phone, no docs, no Trace Explorer.
- **Time it.** Set a 2-hour timer.
- **Don't peek.** Resist the urge to check the answer key until you finish.
- **Answer everything.** Unanswered questions count as wrong.

## Scoring

After finishing:

1. Use `mock-exam-1-answers.md` to grade.
2. Compute per-domain percentages.
3. Pass criterion (self-imposed): **≥80% overall and ≥70% per domain**.
4. If under, run adaptive weak-area drills on the lowest-scoring domain(s) and retake within 3–5 days.
