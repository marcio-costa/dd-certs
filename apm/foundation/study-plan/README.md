# 4-Week Study Plan — APM & Distributed Tracing Fundamentals

> **2 hours per day, 5 days per week, 4 weeks** = **40 hours total** of structured prep.
> Weekends are reserved for hands-on lab time and rest. Adjust as needed; the plan is built to be self-paced.

## Plan philosophy

1. **Read** the matching knowledge-base file (~30 min).
2. **Lab** in a real Datadog trial / sandbox (~45 min).
3. **Quiz** the day's topic (~30 min).
4. **Reflect** in `progress/daily-tracker.md` (~15 min).

Every week ends with a **mini-mock + adaptive drill** on weak topics. Week 4 ends with the **full 75-question mock exam** under timed conditions.

```
00:00 – 00:30   Read the day's knowledge-base file
```
