# Scoring Template

> Use this format every time you take any quiz, drill, mini-quiz, or mock.

```
## [Type — Quiz/Drill/Mini/Mock] — Date: YYYY-MM-DD

Source file: _________________________
Conditions:  ☐ open-book  ☐ closed-book  ☐ timed
Time taken:  __ min  /  Limit: __ min

OVERALL: __ / __  (__%)

Per-domain breakdown:
| Domain | Correct | Total | % |
|--------|---------|-------|---|
| 1 — APM Fundamentals       | __ | __ | __ |
| 2 — Instrumentation        | __ | __ | __ |
| 3 — Insight Discovery      | __ | __ | __ |
| 4 — Troubleshooting        | __ | __ | __ |

Missed questions (Q# → KB section to re-read):
- Q__: KB §____
- Q__: KB §____

Action items:
- [ ] Re-read KB §____
- [ ] Run adaptive drill on _________________
- [ ] Re-run this quiz in __ days
```

## Tips

- Always note **conditions** — open-book scores aren't comparable to closed-book ones.
- **Timed simulations** matter — the real exam has 80 sec/question avg.
- Track which **wrong answer** you picked, not just that you missed; the distractor reveals the misconception.
