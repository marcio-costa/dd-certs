# Quizzes — Reinforcement by Domain

These quizzes are **paired with the study plan** for daily reinforcement after reading each KB file. They are smaller and more focused than the full simulations.

## How to use

1. Read the day's KB file first (don't peek at the answer key).
2. Answer all questions for that day's range.
3. Reveal the answer key at the bottom of each quiz file and review explanations.
4. Anything missed → add to your weakness list in `progress/daily-tracker.md`.

## Difficulty mix per quiz

≈30% easy / ≈50% medium / ≈20% hard. Hard questions usually mix two concepts (e.g., USM + Deployment Tracking) and require eliminating very plausible distractors.

## Note on format

The real exam is **single-answer multiple choice only**. These quizzes match that format exactly — no select-N, no ordering.
