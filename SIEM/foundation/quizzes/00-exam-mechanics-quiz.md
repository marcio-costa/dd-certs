# Quiz 00 — Exam mechanics & scope (10 Q)

> Use this quiz at the end of Day 1. Answers + explanations at the bottom.

## Questions

**1.** How long is the Datadog Cloud SIEM for AWS Fundamentals exam (max seat time)?
A. 60 minutes
B. 90 minutes
C. 120 minutes
D. 180 minutes

**2.** How many questions count toward the score?
A. 50 scored + 10 unscored pretest
B. 60 scored
C. 75 scored + 15 unscored pretest
D. 90 scored

**3.** Which question format is used on this exam?
A. Multiple choice with multiple correct answers
B. Multiple choice with a single correct answer
C. Drag-and-drop ordering
D. Free response / written answer

**4.** Which of the following is OUT of scope for this exam?
A. Cloud SIEM Architecture and Workflow
B. AWS Integration and Log Collection
C. Datadog APM and Distributed Tracing
D. OCSF Data Normalization

**5.** How long is a Datadog Fundamentals certification valid?
A. 1 year
B. 2 years
C. 3 years
D. 5 years

**6.** What is the recommended minimum hands-on experience for ideal candidates?
A. 1 month
B. 3 months
C. 6 months
D. 12 months

**7.** Which of the following is NOT a domain on the exam content outline?
A. Content Pack Implementation and Usage
B. Compliance and Reporting
C. Database Monitoring Configuration
D. Optimization and Best Practices

**8.** Are pretest (unscored) questions identified to the candidate during the exam?
A. Yes, marked with a star
B. No, they are not identified
C. Only after submitting
D. Only if the candidate asks

**9.** Which of the following is included as a recommended learning path in the exam guide?
A. APM Fundamentals Learning Path
B. Cloud Security Engineer — Cloud SIEM Learning Path
C. DBM Fundamentals Learning Path
D. Datadog Agent Internals

**10.** A candidate fails their first attempt. How many retakes are typically allowed within a 180-day window?
A. 1
B. 2
C. 3
D. Unlimited

---

## Answer key

| # | Answer | Explanation |
|---|---|---|
| 1 | **C** | The exam guide states 120 minutes maximum seat time. |
| 2 | **A** | 60 multiple-choice items total: 50 scored + 10 unscored pretest. |
| 3 | **B** | The only question format is multiple choice with a single correct answer; distractors are designed to look plausible. |
| 4 | **C** | The exam guide explicitly lists INFRA and APM as out of scope. |
| 5 | **C** | Datadog Fundamentals certifications are valid for 3 years. |
| 6 | **C** | Minimum 6 months of hands-on Cloud SIEM for AWS production experience. |
| 7 | **C** | Database monitoring is not a Cloud SIEM domain (it's its own certification). |
| 8 | **B** | Pretest items are mixed in with scored items and are not identified to the candidate. |
| 9 | **B** | The exam guide explicitly recommends the Cloud Security Engineer — Cloud SIEM Learning Path and the Attacks & Threat Detection Learning Path. |
| 10 | **C** | Up to 3 retakes per 180-day window. |
