# Simulations — All four formats

This folder contains all 4 simulation formats requested.

| Format | Folder | Volume | Use case |
|---|---|---|---|
| Adaptive weak-area drills | `adaptive-drills/` | 100-question bank tagged by domain & sub-topic + protocol doc | Personalized drilling on the topics you keep missing |
| Domain-specific drills | `domain-drills/` | 7 drills, 115 questions total | End-of-week consolidation, one per domain |
| Full mock exam (75 Q / 2h) | `mock-exam/` | 2 full-length mocks (Mock #1 + Mock #2) | Mid- and end-of-program dress rehearsals |
| Daily mini-quiz (10 Q) | `daily-mini/` | 70-question bank with rotation guidance | 10-minute end-of-day warmup, rotated daily |

## Total simulation question volume

| Source | Questions |

Combined with the **220 reinforcement quizzes** in `foundation/quizzes/`, the package gives you **655+ practice questions** from now until exam day.

## Recommended cadence (matches `foundation/study-plan/study-plan.md`)

```
Daily       → Daily Mini-Quiz (10 Q) at end of every study day
End of week → Domain Drill for the domain(s) you covered that week
Day 21      → Full Mock Exam #1
Day 27      → Adaptive Weak-Area Drill + Full Mock Exam #2
Day 28      → Light review only — no full mocks the day before sitting
```
