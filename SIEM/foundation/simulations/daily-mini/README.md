# Daily Mini-Quizzes (10 Qs each)

These are the 10-question warmups used at the end of every study day (per `foundation/study-plan/`). They sample across all 7 domains so by exam day you have rotated through ~280 unique items.

There are **28 days** of mini-quizzes (`day-01.md` … `day-28.md`). To stay fresh:
- Take **only the day's quiz** at the end of the day
- Score honestly (no peeking) — answers are at the bottom of each file
- If you get ≤ 7/10, mark every wrong domain in your `weak-areas.md` log
- The Adaptive Weak-Area Drill (`../adaptive-drills/`) is built to grind exactly those weak areas

Format of each daily file:
1. **10 multiple-choice questions** drawn from the day's domain focus + 2-3 review questions from earlier days
2. Answer key with one-line explanation per question
